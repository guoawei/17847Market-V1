# 週末文創市集報名系統

這是一個用於管理週末文創市集攤商報名的網頁系統。系統提供了完整的報名流程，包括：

## 功能特點

- 攤商報名管理
- 即時通訊系統
- 匯款確認功能
- 報名狀態追蹤
  - 審核中
  - 待付款
  - 確認匯款資訊中
  - 已確認
  - 已過期

## 技術架構

- HTML5
- Tailwind CSS
- Font Awesome
- JavaScript

## 頁面說明

- `vendor-market-detail.html`: 報名詳情頁面
- `vendor-payment.html`: 付款確認頁面
